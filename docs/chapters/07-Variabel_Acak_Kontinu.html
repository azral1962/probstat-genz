8.2 Materi Kuliah: Konsep, Aplikasi, & Komputasi

8.2.1 1. Konsep Dasar

  • Distribusi Normal (Gaussian): Distribusi paling penting, berbentuk lonceng simetris. Ditentukan oleh dua parameter: Mean (\(\mu\)) menentukan pusat, Standar Deviasi (\(\sigma\)) menentukan lebar/sebaran.
    • Standard Normal (Z): Distribusi normal dengan \(\mu=0, \sigma=1\).
  • Distribusi Eksponensial: Memodelkan waktu tunggu antar kejadian dalam proses Poisson. Parameter \(\lambda\) (laju kejadian).
    • Sifat Memoryless: \(P(X > t+s | X > t) = P(X > s)\). Sejarah tidak mempengaruhi masa depan.
  • Probabilitas Kontinu: Peluang pada satu titik spesifik adalah nol. Peluang dihitung sebagai integral dari fungsi kepadatan peluang (PDF).

8.2.2 2. Aplikasi Sistem Informasi

  • SLA (Service Level Agreement): Menjamin bahwa “99% request akan selesai dalam < 300ms”. Ini menggunakan perhitungan ekor distribusi Normal/Log-Normal.
  • MTTF (Mean Time To Failure): Memodelkan umur komponen perangkat keras (hard disk, server) menggunakan distribusi Eksponensial untuk perencanaan maintenance.
  • Anomaly Detection: Menggunakan skor-Z. Jika data poin memiliki \(|Z| > 3\) (jauh dari rata-rata), itu dicurigai sebagai anomali atau serangan siber.

8.2.3 3. Komputasi (Python)

  • Scipy Stats:
    • norm.pdf(x, loc, scale): Tinggi kurva.
    • norm.cdf(x, loc, scale): Luas area di kiri \(x\) (Probabilitas kumulatif).
    • norm.ppf(q, loc, scale): Kebalikan CDF (Mencari nilai \(x\) dari persentase).
  • Visualisasi: Menggunakan seaborn.kdeplot untuk melihat apakah data empiris mendekati bentuk lonceng.

8.4 15 Soal & Solusi

8.4.1 A. Pertanyaan Konseptual

  1. Soal: Mengapa probabilitas \(P(X=x)\) pada distribusi kontinu selalu bernilai nol?
    • Solusi: Karena pada distribusi kontinu, probabilitas didefinisikan sebagai luas area di bawah kurva PDF. Untuk satu titik tunggal (garis vertikal), lebarnya adalah nol, sehingga luas areanya (integral dari \(x\) ke \(x\)) adalah nol. Probabilitas hanya bermakna dalam interval \(P(a < X < b)\).
  2. Soal: Jelaskan makna fisis dari parameter \(\mu\) dan \(\sigma\) pada kurva lonceng distribusi Normal.
    • Solusi:
      • \(\mu\) (Mean): Menunjukkan lokasi pusat atau puncak dari kurva lonceng (nilai yang paling mungkin muncul).
      • \(\sigma\) (Standar Deviasi): Menunjukkan seberapa lebar atau gemuk kurva tersebut. \(\sigma\) besar berarti kurva landai (data menyebar), \(\sigma\) kecil berarti kurva lancip (data terpusat di sekitar mean).
  3. Soal: Apa itu skor-Z dan mengapa kita perlu menstandarisasi variabel acak Normal?
    • Solusi: Skor-Z (\(z=(x-\mu)/\sigma\)) mengukur berapa standar deviasi suatu nilai data berada dari rata-ratanya. Kita menstandarisasi variabel ke Normal Standar (\(Z \sim N(0,1)\)) agar bisa membandingkan data dari distribusi yang berbeda (misal: membandingkan nilai ujian Matematika dan Fisika) dan menggunakan tabel probabilitas standar.
  4. Soal: Jelaskan sifat memoryless (tidak memiliki memori) pada distribusi Eksponensial dalam konteks kegagalan komponen.
    • Solusi: Sifat memoryless berarti peluang komponen akan rusak dalam \(t\) jam ke depan tidak dipengaruhi oleh sudah berapa lama komponen itu bekerja. Komponen bekas (yang masih berfungsi) dianggap memiliki reliabilitas yang sama dengan komponen baru. \(P(X > t+s | X > s) = P(X > t)\).
  5. Soal: Bagaimana hubungan antara distribusi Normal dan Teorema Limit Pusat secara intuitif?
    • Solusi: Teorema Limit Pusat (CLT) menyatakan bahwa jika kita menjumlahkan banyak variabel acak independen (apa pun distribusi aslinya), hasil penjumlahannya akan cenderung berdistribusi Normal. Ini menjelaskan mengapa distribusi Normal muncul di mana-mana di alam (seperti tinggi badan), karena fenomena tersebut adalah hasil akumulasi dari banyak faktor genetik dan lingkungan yang kecil-kecil.

8.4.2 B. Pertanyaan Aplikatif

  1. Soal: Waktu respons server berdistribusi Normal dengan mean 200ms dan standar deviasi 50ms. Berapa persen respons yang memakan waktu lebih dari 300ms?
    • Solusi: Hitung skor-Z: \(Z=(300-200)/50=2\). Cari peluang \(P(Z > 2)\). Dari tabel/rumus, \(P(Z < 2) \approx 0.9772\). Maka \(P(X > 300) = 1 - 0.9772 = 0.0228\) atau 2.28%.
  2. Soal: Umur pakai sebuah baterai laptop berdistribusi Eksponensial dengan rata-rata 5 tahun. Berapa probabilitas baterai bertahan lebih dari 8 tahun?
    • Solusi: Rata-rata (\(\mu\)) = \(1/\lambda=5 \rightarrow \lambda=0.2\). Fungsi reliabilitas (survival): \(P(X > x) = e^{-\lambda x}\). \(P(X > 8) = e^{-0.2 \times 8} = e^{-1.6} \approx 0.2019\) atau 20.19%.
  3. Soal: Skor ujian masuk sebuah universitas berdistribusi Normal dengan rata-rata 500 dan simpangan baku 100. Jika diambil 10% terbaik, berapakah skor minimumnya?
    • Solusi: Kita mencari nilai \(x\) dimana \(P(X > x) = 0.10\), atau \(P(X < x) = 0.90\). Dari tabel Z invers, nilai Z untuk area 0.90 adalah \(\approx 1.28\). \(x = \mu + Z\sigma = 500 + (1.28 \times 100) = 628\). Skor minimum 628.
  4. Soal: Analisis lebar pita (bandwidth) yang digunakan oleh user di mana penggunaan rata-rata adalah 10 Mbps dengan distribusi Normal.
    • Solusi: Karena bandwidth tidak bisa negatif, distribusi Normal murni mungkin kurang tepat jika \(\sigma\) besar (karena ekor kiri bisa masuk area negatif). Namun, sebagai pendekatan, kita bisa menghitung peluang penggunaan melebihi kapasitas tertentu (misal 15 Mbps) untuk capacity planning.
  5. Soal: Gunakan distribusi Eksponensial untuk memodelkan waktu antar-kedatangan (inter-arrival time) permintaan ke server web.
    • Solusi: Jika server menerima rata-rata 5 request per detik (\(\lambda=5\)), maka waktu antar-kedatangan mengikuti Exp(5). Rata-rata waktu tunggu antar request adalah \(1/5=0.2\) detik. Probabilitas menunggu lebih dari 1 detik sangat kecil: \(e^{-5(1)} = 0.0067\).

8.4.3 C. Pertanyaan Komputasional

  1. Soal: Gunakan scipy.stats.norm.cdf untuk menghitung probabilitas di antara dua nilai pada distribusi Normal.
    • Solusi:
    from scipy.stats import norm  
mu, sigma = 100, 15  
# P(85 < X < 115)  
prob = norm.cdf(115, mu, sigma) - norm.cdf(85, mu, sigma)  
print(f"Probabilitas: {prob:.4f}")
  2. Soal: Tulis skrip Python untuk menghasilkan plot distribusi Normal Standar dan arsirlah area di bawah kurva untuk \(P(-1 < Z < 1)\).
    • Solusi:
    import numpy as np; import matplotlib.pyplot as plt  
from scipy.stats import norm  
x = np.linspace(-3, 3, 100)  
plt.plot(x, norm.pdf(x))  
x_fill = np.linspace(-1, 1, 100)  
plt.fill_between(x_fill, norm.pdf(x_fill), alpha=0.5)  
plt.show()
  3. Soal: Implementasikan fungsi untuk menghitung skor-Z dari sekumpulan data numerik menggunakan numpy.
    • Solusi:
    import numpy as np  
def calculate_z_scores(data):  
return (data - np.mean(data)) / np.std(data)  
data = [1, 2, 3, 4, 5]  
print(calculate_z_scores(data))
  4. Soal: Buat simulasi untuk membuktikan sifat memoryless dari distribusi Eksponensial dengan membangkitkan data acak.
    • Solusi:
    from scipy.stats import expon  
data = expon.rvs(scale=10, size=10000) # Mean=10  
# Peluang X > 15  
p_tot = len(data[data > 15]) / len(data)  
# Peluang X > 15 GIVEN X > 5 (sisa 10)  
subset = data[data > 5]  
p_cond = len(subset[subset > 15]) / len(subset)  
# Bandingkan dengan P(X > 10)  
p_ref = len(data[data > 10]) / len(data)  
print(f"P(X>10): {p_ref:.3f}, P(X>15|X>5): {p_cond:.3f}")
  5. Soal: Visualisasikan bagaimana perubahan \(\sigma\) mempengaruhi “keruncingan” kurva Normal menggunakan matplotlib.
    • Solusi:
    x = np.linspace(-10, 10, 200)  
plt.plot(x, norm.pdf(x, 0, 1), label='Sigma=1')  
plt.plot(x, norm.pdf(x, 0, 0.5), label='Sigma=0.5 (Lancip)')  
plt.plot(x, norm.pdf(x, 0, 2), label='Sigma=2 (Landai)')  
plt.legend(); plt.show()

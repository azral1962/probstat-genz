Naive Bayes Claassifier

Prinsip dasar Naive Bayes classifier adalah: mengklasifikasikan objek dengan membandingkan seberapa “mungkin” data itu muncul pada tiap kelas, lalu memilih kelas dengan probabilitas posterior terbesar.

1) Bayes sebagai mesin “update keyakinan”

Naive Bayes memakai Teorema Bayes:

[ P(Cx)P(C),P(xC)]

  • Prior (P(C)): seberapa sering kelas itu terjadi sebelum melihat data (mis. spam cuma 1%).
  • Likelihood (P(xC)): seberapa cocok data (x) jika kelasnya (C).
  • Posterior (P(Cx)): keyakinan setelah melihat data.

Kita tidak perlu menghitung penyebut (P(x)) karena sama untuk semua kelas, jadi cukup membandingkan skor: [ (C)=P(C),P(xC)]

2) Asumsi “naive”: fitur dianggap independen bersyarat pada kelas

Agar (P(xC)) mudah dihitung, Naive Bayes membuat asumsi:

[ P(x_1,x_2,,x_dC)_{j=1}^d P(x_jC)]

Artinya: setelah kelas diketahui, tiap fitur dianggap tidak saling mempengaruhi.

Contoh:

  • Spam detection: kata “gratis”, “promo”, “klik” dianggap independen dengan syarat email itu spam/ham.
  • Credit scoring: fitur seperti DTI, income bin, purpose dianggap independen dengan syarat borrower good/bad.

Ini memang “naive” (di dunia nyata fitur sering berkorelasi), tapi sering tetap bekerja baik.

3) Keputusan klasifikasi: pilih kelas dengan posterior terbesar

[ = _C ]

Biasanya dipakai log supaya:

  • tidak underflow (perkalian angka kecil jadi 0),
  • berubah jadi penjumlahan.

4) Intuisi versi “LLR” (Sherlock version)

Untuk binary class (1 vs 0), yang dibandingkan adalah log-likelihood ratio:

[ =================================

  • _j ]

  • prior memberi “bias awal”

  • tiap fitur memberi “bukti” pro/kon kelas 1

5) Kenapa perlu smoothing (Laplace)

Jika ada fitur yang tidak pernah muncul di kelas tertentu saat training, maka (P(x_jC)=0) dan seluruh skor jadi 0 (log jadi (-)). Solusinya: Laplace smoothing (tambah ()) agar tidak ada probabilitas nol.

6) Kapan Naive Bayes cocok

  • Data teks (spam, sentiment): sangat cepat dan baseline kuat.
  • Data tabular diskret: oke jika fitur tidak terlalu saling terikat.
  • Dataset besar/fitur banyak: bagus karena training sederhana.

1️⃣ Gaussian Naive Bayes

🔹 Asumsi

Setiap fitur numerik mengikuti distribusi Gaussian (Normal) dalam setiap kelas:

[ x_j C ({Cj}, {Cj}^2)]

Likelihood:

[ P(x_j C) =============

(-) ]

🔹 Kapan Dipakai?

✔ Fitur numerik kontinu ✔ Credit scoring (income, DTI, umur, dll) ✔ Sensor data ✔ Fraud detection berbasis angka

🔹 Contoh P2P Lending

Fitur:

  • income (numeric)
  • DTI ratio
  • loan history months

Jika borrower GOOD punya income rata-rata 12 juta dan BAD rata-rata 4 juta, maka income tinggi otomatis meningkatkan likelihood kelas GOOD.

🔹 Kelebihan

  • Tidak perlu binning
  • Cepat
  • Parameter sedikit (mean & variance per fitur per kelas)

🔹 Kekurangan

  • Jika distribusi tidak mendekati normal → performa turun
  • Sensitif terhadap outlier

2️⃣ Multinomial Naive Bayes

🔹 Asumsi

Fitur adalah count/frequency (biasanya teks)

[ P(x C) = _j P(w_j C)^{count(w_j)}]

🔹 Kapan Dipakai?

✔ Spam detection ✔ Sentiment analysis ✔ Document classification ✔ Text mining

🔹 Contoh Spam

Kata:

  • “gratis” sangat sering muncul di spam
  • “rapat” jarang muncul di spam

LLR untuk kata “gratis”:

[ ]

Jika besar → kuat mendukung spam.

🔹 Kelebihan

  • Sangat efektif untuk teks
  • Training sangat cepat
  • Tahan terhadap dimensi tinggi

🔹 Kekurangan

  • Asumsi independensi kata tidak realistis
  • Tidak mempertimbangkan urutan kata (kecuali n-gram)

3️⃣ Bernoulli Naive Bayes

🔹 Asumsi

Fitur adalah binary (0/1) → hanya memperhatikan apakah fitur muncul atau tidak.

[ P(x_jC)=p{x_j}(1-p){1-x_j}]

🔹 Kapan Dipakai?

✔ Text classification (presence/absence) ✔ Fitur boolean ✔ Sistem rule-based sederhana

🔹 Perbedaan dengan Multinomial

Multinomial Bernoulli
Menghitung frekuensi Hanya hadir/tidak
Bagus untuk teks panjang Bagus untuk teks pendek
Umum dipakai di NLP Lebih jarang

🎯 Perbandingan Singkat

Tipe Data Model
Numerik kontinu GaussianNB
Count teks MultinomialNB
Binary/boolean BernoulliNB
Kategori diskret CategoricalNB

🔬 Intuisi Unified (Very Important)

Semua varian adalah bentuk khusus dari:

Perbedaannya hanya pada bentuk (P(x_jC)).

import re
from collections import Counter

def tokenize(text):
    # Lowercase + ambil kata
    return re.findall(r"[a-zA-Z0-9]+", text.lower())

def train_keyword_model(emails, labels):
    """
    emails: list[str]
    labels: list[int]  # 1 spam, 0 ham
    return counts kata per kelas + prior
    """
    spam_words = Counter()
    ham_words = Counter()
    n_spam = 0
    n_ham = 0

    for text, y in zip(emails, labels):
        words = tokenize(text)
        if y == 1:
            n_spam += 1
            spam_words.update(words)
        else:
            n_ham += 1
            ham_words.update(words)

    prior_spam = n_spam / (n_spam + n_ham) if (n_spam + n_ham) else 0.0
    return {"spam_words": spam_words, "ham_words": ham_words, "prior_spam": prior_spam}

def predict_naive_keyword(text, model, keywords=("gratis", "promo", "diskon"), threshold=1):
    """
    SANGAT naive: jika jumlah keyword muncul >= threshold => spam
    """
    words = tokenize(text)
    c = Counter(words)
    score = sum(c[k] for k in keywords)
    return 1 if score >= threshold else 0, score

import numpy as np

def confusion_2x2(y_true, y_pred, positive_label=1):
    y_true = np.asarray(y_true)
    y_pred = np.asarray(y_pred)

    TP = int(np.sum((y_true == positive_label) & (y_pred == positive_label)))
    TN = int(np.sum((y_true != positive_label) & (y_pred != positive_label)))
    FP = int(np.sum((y_true != positive_label) & (y_pred == positive_label)))
    FN = int(np.sum((y_true == positive_label) & (y_pred != positive_label)))

    cm = np.array([[TN, FP],
                   [FN, TP]])
    return cm, TP, FP, TN, FN

def metrics_from_cm(TP, FP, TN, FN):
    precision = TP / (TP + FP) if (TP + FP) else 0.0
    recall    = TP / (TP + FN) if (TP + FN) else 0.0
    accuracy  = (TP + TN) / (TP + TN + FP + FN) if (TP+TN+FP+FN) else 0.0
    f1 = (2*precision*recall)/(precision+recall) if (precision+recall) else 0.0
    return {"precision": precision, "recall": recall, "accuracy": accuracy, "f1": f1}

A1) Naive Bayes untuk teks (Multinomial NB) — core engine

Dipakai untuk spam detection dan sentiment analysis.

import re
from collections import Counter, defaultdict

def tokenize(text):
    return re.findall(r"[a-zA-Z0-9]+", text.lower())

class MultinomialNB_Text_NoSK:
    """
    Multinomial Naive Bayes untuk teks:
    P(class|doc) ∝ P(class) * Π P(word|class)^{count(word,doc)}
    dengan Laplace smoothing.
    """
    def __init__(self, alpha=1.0):
        self.alpha = float(alpha)
        self.class_priors_log = {}
        self.word_logprob = {}   # dict[class] -> dict[word] -> log P(word|class)
        self.vocab = set()
        self.classes_ = None

    def fit(self, texts, y):
        y = list(y)
        texts = list(texts)
        classes = sorted(set(y))
        self.classes_ = classes

        # Count words per class
        word_counts = {c: Counter() for c in classes}
        class_counts = Counter(y)

        for text, label in zip(texts, y):
            words = tokenize(text)
            word_counts[label].update(words)
            self.vocab.update(words)

        V = len(self.vocab)

        # log priors
        n = len(y)
        self.class_priors_log = {c: np.log(class_counts[c] / n) for c in classes}

        # log likelihoods with Laplace smoothing
        self.word_logprob = {}
        for c in classes:
            total_words_in_class = sum(word_counts[c].values())
            denom = total_words_in_class + self.alpha * V
            self.word_logprob[c] = {}
            for w in self.vocab:
                num = word_counts[c][w] + self.alpha
                self.word_logprob[c][w] = np.log(num / denom)

            # store unseen word logprob (for words outside vocab during predict)
            self.word_logprob[c]["<UNK>"] = np.log(self.alpha / denom)

        return self

    def predict_proba(self, texts):
        probs = []
        for text in texts:
            words = tokenize(text)
            counts = Counter(words)
            log_scores = []
            for c in self.classes_:
                s = self.class_priors_log[c]
                for w, k in counts.items():
                    s += k * self.word_logprob[c].get(w, self.word_logprob[c]["<UNK>"])
                log_scores.append(s)

            # softmax
            log_scores = np.array(log_scores)
            log_scores = log_scores - np.max(log_scores)
            p = np.exp(log_scores)
            p = p / p.sum()
            probs.append(p)
        return np.array(probs)

    def predict(self, texts):
        proba = self.predict_proba(texts)
        idx = np.argmax(proba, axis=1)
        return np.array([self.classes_[i] for i in idx])

A4) (3) Credit Scoring P2P Lending — tanpa sklearn (Discrete Naive Bayes)

Untuk tabular, cara paling aman tanpa sklearn: diskritisasi fitur numerik → lalu Naive Bayes diskret (kategori).

class DiscreteNB_NoSK:
    """
    Naive Bayes untuk fitur diskret/kategori.
    P(y|x) ∝ P(y) * Π P(x_j = v | y)
    Laplace smoothing per fitur.
    """
    def __init__(self, alpha=1.0):
        self.alpha = float(alpha)
        self.classes_ = None
        self.prior_log = {}
        self.cond_log = {}   # cond_log[y][j][value] = log prob
        self.feature_values_ = []  # list of sets

    def fit(self, X, y):
        X = np.asarray(X, dtype=object)
        y = np.asarray(y, dtype=object)
        n, d = X.shape
        classes = sorted(set(y))
        self.classes_ = classes

        # nilai unik per fitur
        self.feature_values_ = [set(X[:, j]) for j in range(d)]

        # prior
        counts_y = Counter(y)
        self.prior_log = {c: np.log(counts_y[c] / n) for c in classes}

        # conditional counts
        self.cond_log = {c: [defaultdict(float) for _ in range(d)] for c in classes}

        for c in classes:
            Xc = X[y == c]
            for j in range(d):
                vals = list(self.feature_values_[j])
                counts = Counter(Xc[:, j])
                V = len(vals)
                denom = len(Xc) + self.alpha * V
                for v in vals:
                    num = counts.get(v, 0) + self.alpha
                    self.cond_log[c][j][v] = np.log(num / denom)
                # unseen:
                self.cond_log[c][j]["<UNK>"] = np.log(self.alpha / denom)

        return self

    def predict_proba(self, X):
        X = np.asarray(X, dtype=object)
        n, d = X.shape
        probs = np.zeros((n, len(self.classes_)), dtype=float)

        for i in range(n):
            log_scores = []
            for c in self.classes_:
                s = self.prior_log[c]
                for j in range(d):
                    v = X[i, j]
                    s += self.cond_log[c][j].get(v, self.cond_log[c][j]["<UNK>"])
                log_scores.append(s)

            log_scores = np.array(log_scores)
            log_scores = log_scores - np.max(log_scores)
            p = np.exp(log_scores)
            p = p / p.sum()
            probs[i, :] = p
        return probs

    def predict(self, X):
        proba = self.predict_proba(X)
        idx = np.argmax(proba, axis=1)
        return np.array([self.classes_[i] for i in idx])

def bin_income(x):
    # juta rupiah/bulan (contoh)
    if x < 5: return "low"
    if x < 12: return "mid"
    return "high"

def bin_dti(x):
    # debt-to-income ratio (0..1)
    if x < 0.2: return "low"
    if x < 0.5: return "mid"
    return "high"

def bin_history_months(m):
    if m < 6: return "short"
    if m < 24: return "mid"
    return "long"

import numpy as np

def logit(p):
    p = float(p)
    p = min(max(p, 1e-15), 1-1e-15)
    return np.log(p/(1-p))

def sigmoid(z):
    return 1/(1+np.exp(-z))

A1) LLR untuk TEKS (Multinomial NB dengan counts)

LLR dokumen untuk kelas 1 vs 0:

[ (d)=+_w c(w,d)]

Engine: fit → score_llr → predict

import re
from collections import Counter

def tokenize(text):
    return re.findall(r"[a-zA-Z0-9]+", text.lower())

class TextLLR_NoSK:
    """
    Multinomial NB versi LLR untuk binary classification.
    label 1 = positive/spam, label 0 = negative/ham
    """
    def __init__(self, alpha=1.0):
        self.alpha = float(alpha)
        self.vocab = set()
        self.log_prior_odds = 0.0
        self.log_pw_given_1 = {}
        self.log_pw_given_0 = {}
        self.log_ratio_w = {}   # log P(w|1) - log P(w|0)
        self.log_unk_ratio = 0.0

    def fit(self, texts, y):
        texts = list(texts)
        y = np.asarray(y).astype(int)

        # prior odds
        p1 = y.mean()
        self.log_prior_odds = logit(p1)

        # count words per class
        c1 = Counter()
        c0 = Counter()
        n1 = 0
        n0 = 0
        for t, label in zip(texts, y):
            words = tokenize(t)
            if label == 1:
                c1.update(words); n1 += 1
            else:
                c0.update(words); n0 += 1
            self.vocab.update(words)

        V = len(self.vocab)

        total1 = sum(c1.values())
        total0 = sum(c0.values())

        denom1 = total1 + self.alpha * V
        denom0 = total0 + self.alpha * V

        # precompute log probs and log ratios
        self.log_pw_given_1 = {}
        self.log_pw_given_0 = {}
        self.log_ratio_w = {}

        for w in self.vocab:
            p_w_1 = (c1[w] + self.alpha) / denom1
            p_w_0 = (c0[w] + self.alpha) / denom0
            self.log_pw_given_1[w] = np.log(p_w_1)
            self.log_pw_given_0[w] = np.log(p_w_0)
            self.log_ratio_w[w] = self.log_pw_given_1[w] - self.log_pw_given_0[w]

        # unseen word ratio
        p_unk_1 = self.alpha / denom1
        p_unk_0 = self.alpha / denom0
        self.log_unk_ratio = np.log(p_unk_1) - np.log(p_unk_0)
        return self

    def score_llr(self, text):
        counts = Counter(tokenize(text))
        s = self.log_prior_odds
        for w, k in counts.items():
            s += k * self.log_ratio_w.get(w, self.log_unk_ratio)
        return float(s)

    def predict(self, texts, threshold=0.0):
        scores = np.array([self.score_llr(t) for t in texts], dtype=float)
        return (scores > threshold).astype(int), scores

    def predict_proba(self, texts, threshold=0.0):
        # P(y=1|x) = sigmoid(LLR - threshold_shift)
        scores = np.array([self.score_llr(t) for t in texts], dtype=float)
        z = scores - threshold
        p1 = sigmoid(z)
        return np.vstack([1-p1, p1]).T, scores

Untuk fitur kategori/diskret:

[ (x)=+_j ]

Engine: fit dari X kategori

from collections import Counter, defaultdict

class DiscreteLLR_NoSK:
    """
    Binary discrete NB, output skor LLR.
    y=1: GOOD (lancar), y=0: BAD (default)
    """
    def __init__(self, alpha=1.0):
        self.alpha = float(alpha)
        self.log_prior_odds = 0.0
        self.feature_vals = []
        self.log_ratio = []      # list[dict[val] -> log P(val|1)-log P(val|0)]
        self.log_unk_ratio = []  # per fitur

    def fit(self, X, y):
        X = np.asarray(X, dtype=object)
        y = np.asarray(y).astype(int)
        n, d = X.shape

        self.log_prior_odds = logit(y.mean())
        self.feature_vals = [sorted(set(X[:, j])) for j in range(d)]
        self.log_ratio = [dict() for _ in range(d)]
        self.log_unk_ratio = [0.0 for _ in range(d)]

        X1 = X[y == 1]
        X0 = X[y == 0]

        for j in range(d):
            vals = self.feature_vals[j]
            V = len(vals)

            c1 = Counter(X1[:, j]) if len(X1) else Counter()
            c0 = Counter(X0[:, j]) if len(X0) else Counter()

            denom1 = len(X1) + self.alpha * V
            denom0 = len(X0) + self.alpha * V

            for v in vals:
                p1 = (c1.get(v, 0) + self.alpha) / denom1
                p0 = (c0.get(v, 0) + self.alpha) / denom0
                self.log_ratio[j][v] = np.log(p1) - np.log(p0)

            # unseen ratio
            self.log_unk_ratio[j] = np.log(self.alpha/denom1) - np.log(self.alpha/denom0)

        return self

    def score_llr_row(self, row):
        s = self.log_prior_odds
        for j, v in enumerate(row):
            s += self.log_ratio[j].get(v, self.log_unk_ratio[j])
        return float(s)

    def predict(self, X, threshold=0.0):
        X = np.asarray(X, dtype=object)
        scores = np.array([self.score_llr_row(r) for r in X], dtype=float)
        yhat = (scores > threshold).astype(int)
        return yhat, scores

    def predict_proba(self, X, threshold=0.0):
        yhat, scores = self.predict(X, threshold=threshold)
        p1 = sigmoid(scores - threshold)
        return np.vstack([1-p1, p1]).T, scores

B) DENGAN sklearn (LLR dari parameter NB)

Intinya: sklearn sudah menghitung joint log-likelihood per kelas. Untuk binary, LLR = (log joint kelas 1) − (log joint kelas 0).

B1) LLR untuk teks dengan sklearn (CountVectorizer + MultinomialNB)

import numpy as np
from sklearn.feature_extraction.text import CountVectorizer
from sklearn.naive_bayes import MultinomialNB

def llr_text_sklearn(texts_train, y_train, texts_test, alpha=1.0):
    vec = CountVectorizer(lowercase=True)
    Xtr = vec.fit_transform(texts_train)

    nb = MultinomialNB(alpha=alpha)
    nb.fit(Xtr, y_train)

    Xte = vec.transform(texts_test)

    # joint log-likelihood per kelas: shape (n_samples, 2)
    jll = nb._joint_log_likelihood(Xte)

    # asumsikan kelas bernilai [0,1]
    # LLR = log P(x,y=1) - log P(x,y=0)
    llr = jll[:, 1] - jll[:, 0]
    pred = (llr > 0).astype(int)

    # opsional: probabilitas via sigmoid(llr) (untuk binary NB, ini konsisten)
    p1 = sigmoid(llr)
    proba = np.vstack([1-p1, p1]).T

    return pred, llr, proba, vec, nb

import numpy as np
from sklearn.naive_bayes import CategoricalNB
from sklearn.preprocessing import OrdinalEncoder

def llr_credit_categorical_sklearn(X_train, y_train, X_test, alpha=1.0):
    enc = OrdinalEncoder(handle_unknown="use_encoded_value", unknown_value=-1)
    Xtr_enc = enc.fit_transform(X_train)
    Xte_enc = enc.transform(X_test)

    nb = CategoricalNB(alpha=alpha)
    nb.fit(Xtr_enc, y_train)

    jll = nb._joint_log_likelihood(Xte_enc)
    llr = jll[:, 1] - jll[:, 0]
    pred = (llr > 0).astype(int)

    p1 = sigmoid(llr)
    proba = np.vstack([1-p1, p1]).T
    return pred, llr, proba, enc, nb
